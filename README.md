# Weather-App

Описание: Проект дает возможность посмотреть погоду любового города с помошью библиотеки (openweathermap)

## Технологии
- React
- Type-Script
- Css
- HTML

## Установка
Скачать файл с GitHub
Открыть с помощью редактора кода 
В консоле запустить LocalHost через команду 
``
npm run dev
``
